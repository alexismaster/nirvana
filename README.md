# Nirvana

PHP Micro Framework


# Install

1. `git clone https://github.com/alexismaster/nirvana.git`
2. `git checkout develop`
3. Настроить вебсервер так чтобы `DOCUMENT_ROOT` указывал на `path/to/nirvana/public`
